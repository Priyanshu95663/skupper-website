Using Skupper with Minikube

Prerequisites

Skupper requires cluster-external network access in order to form connections between clusters. Run minikube tunnel in the background to enable this access.

minikube tunnel

Logging in

Minikube does not offer a dedicated login command, but you can re-run the minikube start command in each console you wish to configure.

Console for US East
export KUBECONFIG=$HOME/.kube/config-us-east
minikube start
Console for EU North
export KUBECONFIG=$HOME/.kube/config-eu-north
minikube start
