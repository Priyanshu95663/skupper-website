Multi-cluster communication for Kubernetes

Skupper is a layer 7 service interconnect. It enables secure communication across Kubernetes clusters with no VPNs or special firewall rules.

With Skupper, your application is not confined to one cluster. It can span multiple cloud providers, data centers, and regions.
